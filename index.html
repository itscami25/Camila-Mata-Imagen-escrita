La asignatura de imagen escrita se enfoca en la unión entre diseñar, programar y generar, trayendo a la vida formas de repetición qué solo podríamos llegar a imaginar, haciendo que estas ideas se vuelvan visibles en este ámbito virtual.

Este trabajo esta hecho a base de p5.js y github, siendo los traductores e intermediarios para la correcta ejecución de cada código utilizado para esta web.

Este curso nos enseñó que el pensamiento humano puede convivir con el pensamiento computacional, llegando a ser una buena herramienta para el diseñador, entregándole nuevas herramientas para nuevas posibilidades creativas.
